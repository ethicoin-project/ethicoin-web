# ETHICOIN Moderator Application Form

Submit your application to the ETHICOIN moderation team via Telegram or email. Use the template below.

## Recruitment Timeline

```
Week 1: Post recruitment announcement
Week 1-2: Collect applications
Week 2: Interview top candidates (1-on-1)
Week 2: Onboard selected moderators
Week 3: Training period (paired with senior mod)
Week 4: Independent moderation begins
```
