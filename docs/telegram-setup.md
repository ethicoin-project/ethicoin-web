# ETHICOIN Telegram Official Channel Setup

## Channel Structure

### Main Channels
```
1. @EthicoinCore — Official Announcements
   - News, updates, integrations
   - Admin-only posts
   - 1-2 posts per day

2. @EthicoinCore_Chat — Community Discussion
   - General discussion
   - Price talk
   - Technical questions
   - Off-topic (limited)

3. @EthicoinCore_Support — Support & Help
   - Technical support
   - Account issues
   - Bug reports
   - Response SLA: 24 hours

4. @EthicoinCore_TR — Turkish Community (Optional)
   - Content in Türkçe
   - Turkish community focus
   - Weekly meetups/AMAs
```

---

## Channel Moderation Setup

### Moderator Team
```
- Head Moderator (1)
- Senior Mods (3-5)
- Junior Mods (5-10)
- Community Managers (2-3)
- Language Mods (Turkish, etc.)
```

### Moderation Rules
```
1. No spam or self-promotion (except in #promotions)
2. No FUD, hype, or price manipulation
3. No referral links or affiliate schemes
4. No private/paid groups promotion
5. No hate speech, racism, or discrimination
6. Max 3 consecutive messages
7. English or Turkish only (in respective channels)
```

### Auto-Moderation Bot Setup
```
Command: /start
Shows: Welcome message with rules & links

Command: /price
Shows: Current ETHIC price (DexScreener API)

Command: /holders
Shows: Unique holders count (BlockScout API)

Command: /help
Shows: Available commands list

Command: /rules
Shows: Community guidelines

Command: /links
Shows: Important links (website, contract, etc.)
```
