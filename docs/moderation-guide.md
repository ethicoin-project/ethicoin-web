# ETHICOIN Community Moderation Handbook

## Moderator Code of Conduct

### Core Values
- **Integrity:** Fair, consistent, impartial moderation
- **Professionalism:** Represent ETHICOIN with respect
- **Transparency:** Explain moderation decisions
- **Patience:** Handle conflicts calmly
- **Growth:** Foster positive community culture

---

## Moderation Levels & Warnings

### Level 1: Warning (First Offense)
```
Offense:
- Mild spam (2-3 messages)
- Off-topic in announcements
- Minor rule violation
- Accidental FUD

Action:
- Send private warning message
- Explain rule clearly
- Give 24-hour grace period
- No public removal
```

### Level 2: Mute (Second Offense)
```
Offense:
- Repeated spam after warning
- Aggressive off-topic behavior
- Continued FUD spreading
- Referral link posting

Action:
- Mute for 24 hours
- Explain consequences
- Suggest reading #rules
- Track repeat offenders
```

### Level 3: Temporary Ban (Serious/Repeated)
```
Offense:
- Hate speech, discrimination
- Coordinated FUD campaign
- Scam promotion
- Threats or harassment
- 3+ level 2 violations

Action:
- 7-day ban
- Detailed ban reason message
- Appeal process available
- Log in moderation database
```

### Level 4: Permanent Ban
```
Offense:
- Hate speech or racism
- Threats or doxxing
- Scam/ponzi scheme
- Repeated after appeals
- Criminal activity

Action:
- Immediate permanent removal
- Ban from all ETHICOIN channels
- Report to Telegram if needed
- No appeal (final)
```

## Moderation Tools & Automation

### Telegram Bot Features
```
/mute @user [reason] — Mute user for 24h
/ban @user [reason] — Ban user permanently
/warn @user [reason] — Send warning
/appeal — Allow banned user to appeal
/report @user [reason] — Report to senior mods
/stats — Show moderation statistics
```
